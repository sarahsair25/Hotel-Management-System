# Hotel-Management-System (Python)
Python CLI Hotel Management System


A menu-driven Hotel Management System built using Python and Object-Oriented Programming.

## Features
- User Registration
- Menu-driven CLI interface
- Input validation
- Persistent storage using JSON
- Password handling
- Object-Oriented Design

## Technologies Used
- Python
- OOP Concepts
- File Handling (JSON)
